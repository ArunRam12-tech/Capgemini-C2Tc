# Capgemini-C2Tc

This repository contains Java programs and practice exercises completed during the Capgemini C2TC (Campus to Corporate) training program.

## Topics Covered

- Java Basics
- Classes and Objects
- Constructors
- Access Modifiers
- Inheritance
- Encapsulation
- Method Overriding
- Polymorphism
- Packages
- Object-Oriented Programming (OOP) Concepts

## Technologies Used

- Java
- Eclipse IDE
- Git
- GitHub

## Purpose

The purpose of this repository is to practice Java programming concepts and build a strong foundation in Object-Oriented Programming as part of the Capgemini C2TC training program.
